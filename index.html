Smart Print Orientation Analyzer

Using Print Compass

Print Compass analyzes STL geometry to help you find stronger and easier-to-print orientations before slicing.

Camera Controls
• Left drag — Orbit the camera
• Right drag — Pan across the build plate
• Ctrl + mouse wheel — Zoom
Manual Rotation
Rotate the model with X / Y / Z buttons.
Use the Step selector to control rotation precision.
Orientation Analysis
Analyze Orientation samples many possible rotations and suggests:
• Best Balanced
• Lowest Support
• Best Stability
View Modes
Toggle between Strength, Supports, and Overhang visualization modes.
Overhang Heatmap
Highlights downward-facing surfaces likely to need support.
Optional Support Shadows project likely support columns to the build plate.
Candidate Browser
Cycle through the top ranked orientations and inspect their strength and support scores.
STL Inspector
Detects mesh problems such as holes, non-manifold edges, multiple shells, and degenerate triangles.
🔒 Local Processing Only
All analysis runs locally in your browser. STL files are never uploaded and no model data leaves your device.
Typical Workflow
  1. Load an STL file
  2. Run Analyze Orientation
  3. Preview candidates or recommendations
  4. Inspect overhangs if needed
  5. Export the final orientation
